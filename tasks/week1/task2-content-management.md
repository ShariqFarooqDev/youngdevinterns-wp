## ğŸ“ Key Learnings

### WordPress Content Types:
1. **Posts** - Dynamic, time-based content with categories/tags
2. **Pages** - Static content without categories/tags

### Content Editor (Gutenberg):
- Block-based editor
- Different block types (paragraph, heading, image, etc.)
- Reusable blocks
- Block patterns

### Categories vs Tags:
- **Categories:** Hierarchical, broad topics
- **Tags:** Non-hierarchical, specific keywords

### Homepage Settings:
- Latest posts (blog style)
- Static page (traditional website)
