# Screenshots Directory

This directory contains screenshots of completed tasks for documentation and social media posts.

## Structure

- `week1/` - Screenshots from Week 1 tasks
- `week2-3/` - Screenshots from Week 2-3 tasks
- `week4/` - Screenshots from Week 4 tasks

## Guidelines

- Use descriptive filenames (e.g., `task1-wordpress-dashboard.png`)
- Include before/after screenshots when applicable
- Optimize images for web (compress if needed)
- Use screenshots in social media posts
